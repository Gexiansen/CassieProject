# CassieProject Roadmap

## 当前阶段

本地优先基础、“预算—目标—复盘”决策闭环和正式专项已经建立；当前已完成一轮信息架构与交互减法，进入真实使用观察期，先验证日常记账、预算和月结是否足够顺手，再继续增加专项经验复用功能。

## 已完成

- 完成 `docs/index.html` 的静态代码审查。
- 明确家庭共享需要独立身份、权限、同步和冲突处理，不能直接以 `localStorage` 作为共享数据源。
- 建立项目规范和路线图。
- 将金额改为整数分。
- 使用 UUID 标识账目。
- 增加数据结构版本、校验和旧数据迁移。
- 修复专项详情统计范围和日历年／月状态混用。
- 增加账目编辑、删除撤销和安全导入。
- 移除动态内联事件处理器，完善存储错误反馈。
- 将项目规范统一为 `AGENTS.md`，主账本页面统一为静态站点入口 `docs/index.html`。
- 增加 schema v2 演示账本 `test-data/account-book-demo-v2.json`，用于导入和界面效果测试。
- 增加 GitHub Pages PWA 资源：manifest、Service Worker 和本地应用图标。
- 移除 Google Fonts，`docs/` 运行时不再依赖第三方网络资源。
- 增加备份时间记录、未备份／过期提醒和导入数据摘要。
- 增加异常数据写入锁定、原始内容下载和有效记录重建流程。
- 增加最近使用和本地高频分类；旧版收藏数据继续兼容读取和备份，但不再要求用户手动维护收藏。
- 增加本地快速模板，可保存分类、金额和备注并从首页直接套用。
- 增加复制最近一笔、复制明细账目和“保存并再记”。
- 增加关键词、收支、主分类、子分类、日期和金额组合筛选，并展示结果汇总。
- 增加返回本月快捷入口，筛选结果自动展开到明细。
- 增加完整备份 v1，统一导出账目、最近分类、收藏分类和快速模板。
- 增加“合并账目”“覆盖账目”和“完整恢复账目与设置”，继续兼容 schema v2 和旧数组备份。
- 为快捷设置增加独立版本和更新时间，设置变化后会触发重新备份提醒。
- 增加 decisions v1 独立规划数据，按月份保存总预算和分类预算。
- 增加“规划”页面、概览预算执行卡片、80％预警、100％超支和复制上月预算。
- 将完整备份升级到 v2，纳入规划数据并继续兼容完整备份 v1。
- 增加财务目标，支持应急储备、旅行、教育、偿债和自定义类型。
- 增加独立的目标投入记录、累计进度、剩余金额和按目标日期计算的建议月投入。
- 增加目标编辑、暂停、恢复、完成和重新启用，目标及投入随规划数据进入完整备份。
- 增加月度复盘，汇总收入、支出、结余、预算执行和目标投入，并生成基于本地数据的观察。
- 将月度复盘收敛为“一个结论＋下个月一件事”；旧版额外行动继续兼容保留，上月行动会在次月自动出现并可标记完成。
- 禁止提前填写未来月份复盘，复盘和行动状态随规划数据进入完整备份。
- 增加目标投入明细，支持查看每笔投入的日期、金额和备注。
- 增加目标投入编辑和二次确认撤销，修改后同步更新目标进度与对应月份复盘，并在保存失败时回滚。
- 增加最近三个完整月份的现金流能力分析，排除尚未结束的本月数据。
- 将房租、水电燃气识别为固定承诺，计算固定承诺、可调整支出和月均结余，并提供保守 50％、平衡 70％、激进 90％三种目标投入情景。
- 联合本月预算、历史月均收入和进行中目标的建议月投入，提示计划后的现金流余额或缺口。
- 增加具有稳定 ID 的正式专项，支持旅行、装修、节日、医疗、教育、搬家和其他类型，并记录预算、日期、参与人数和进行中／完成状态。
- 增加当前专项，新建首个专项后自动设为当前，记支出时默认带入；专项完成后自动取消当前状态并停止默认关联。
- 保留住宿、交通、餐饮等支出分类，同时按专项汇总总支出、预算执行、记录数、人均和人均每天费用。
- 将正式专项支出从日常总预算和分类预算中排除，避免生活事件挤占日常消费额度；总收支统计仍包含全部支出。
- 将开始日期位于本月的专项预算纳入现金流计划，并在完整备份 v2 中保存专项定义、当前专项和账目关联。
- 继续兼容旧版专项标签和不含专项字段的旧备份；旧记录不自动改写，新记录使用 `projectId` 关联正式专项。
- 修正当月和当年同比口径：当前周期只与上月／上年同日之前的数据比较，历史周期继续比较完整周期。
- 将一级导航收敛为“首页、明细、计划”，日历合并进明细，月度复盘与现金流判断合并进“月结”。
- 简化记账表单：优先显示最近／高频分类，日期、备注、专项和次要保存方式按需展开；全局只保留一个“记一笔”入口。
- 简化首页和数据管理：首页只保留快速记账、预算、当前专项、一个观察、趋势和排行，完整备份与导入统一放入数据管理弹窗。
- 简化预算和专项指标：分类预算改为可选展开项，只有旅行专项显示人均和人均每天，其他专项显示记录数、周期和剩余预算。
- 允许页面缩放，并将主要导航、分类选择、弹窗操作、记录菜单和删除撤销等高频触控目标调整到至少 44px。

## 进行中

- 观察精简后的日常记账、预算和月结流程，记录真实使用中的阻力，确认高频路径稳定后再开始专项历史对比。

## 待办

- 增加同类型专项历史对比和最近一次参考。
- 比较预算、实际总额、主要分类、人均和人均每天费用。
- 在新建专项时提供历史参考，但不自动替用户决定预算。

### 阶段六：个人自定义与家庭汇总基础

- 将支出分类改为具有稳定 ID 的可管理数据；正式专项已完成稳定 ID 改造。
- 只有明确需要家庭汇总时，再增加成员标识和手动文件合并规则。

### 暂缓

- 家庭资产、负债和净资产快照按当前使用优先级暂不建设。
